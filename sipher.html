Sipher

Sipher is a program I developed using Java Swing to encode and decode messages. It currently supports two ciphers, two secure hashing algorithms, and has a password/code generator


  • Sipher can perform
  • Rotation/Caesar Cipher
  • Vignere Table Cipher
  • SHA-1
  • SHA-256

    • Find this project on github to use it or make contributions (Run the .jar as an executable)!
