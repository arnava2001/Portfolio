Operation Run: December 2018

Operation Run is a game that I designed using Unity for my senior capstone project at Fort Worth Country Day School. I wanted to conduct some research on using videogames as educational platforms in the classroom as videogames are a big interest of mine. Alongside this research, I also created operation run which is a sidescroller that prompts a randomly generated multiplication, addition, or subtraction problem when the user comes into contact with an obstacle. The player has 3 lives and loses one whenever they come into contact with an obstacle or lose all of their remaining lives if they get a question incorrectly.

Find this project on github to play or make contributions!
